# MSH-Character-Generator
The MSH Character Generator, or MSHCG for short, is designed to allow players of the classic Marvel Superheroes RPG by TSR to create new characters as quickly as possible using virtually the same method as described in the Ultimate Powers Book and Players Handbook of the RPGs. This program is not meant to replace the UPB, rather it is meant to be used with the UPB. You still need the rulebooks and UPB for the Power descriptions and other useful information that is not provided by this program. The rulebooks and UPB can be found in pdf format on the internet with a Google search. Also, this is not meant to completely generate a new character. This program essentially provides a quick and easy way to create the foundation, or structure, for the character that you then flesh-out after the character sheet has been saved.

# MSH Character Generator

**Version 2.00.00**
By TaskmasterX

## Overview

MSH Character Generator (MSHCG) is a desktop application designed to streamline character creation for the classic Marvel Super Heroes Role-Playing Game by TSR.

The program follows the character generation process described in the Ultimate Powers Book (UPB) and Player's Handbook, allowing players to quickly generate a character's physical form, abilities, powers, talents, contacts, and weaknesses. Rather than replacing the official rulebooks, MSHCG serves as a companion tool that automates the random generation process and provides a foundation for creating detailed characters.

## Features

* Generate characters using classic Marvel Super Heroes RPG rules
* Randomly determine Physical Form and Origin of Power
* Roll FASERIP abilities, Resources, and Popularity
* Generate powers and power ranks based on the Ultimate Powers Book
* Create talents, contacts, and weaknesses
* Purchase additional powers, talents, and contacts using Resource ranks
* Save completed character sheets as text files
* Optional support for standard rank scores

## How It Works

The character generation process is divided into several stages:

1. **Physical Form** – Determine the character's physical type and origin.
2. **Abilities** – Roll FASERIP statistics, Resources, and Popularity.
3. **Powers** – Generate powers, ranks, bonus powers, and weaknesses.
4. **Talents** – Determine talents and areas of expertise.
5. **Contacts** – Generate professional and personal contacts.
6. **Character Details** – Add name, identity, age, affiliations, and other background information before saving.

## Requirements

This application is intended for use alongside the official Marvel Super Heroes RPG rulebooks and the Ultimate Powers Book. Power descriptions, game mechanics, and supplemental information are not included.

## Technologies Used

* Python
* PyQt5
* Randomized character generation algorithms
* Text file export
